# Sleep-Health-and-Lifestyle-Dataset-Analysis
Overview

This project explores the Sleep, Health, and Lifestyle dataset from Kaggle to analyze how demographic, health, and lifestyle factors influence sleep quality. SQL was used for data exploration and summarizing insights.

Dataset

The dataset contains survey responses on sleep habits, demographics, health indicators, and lifestyle behaviors.

Source: https://www.kaggle.com/code/tanshihjen/eda-sleep-health-and-lifestyle-dataset/input

Purpose

Explore sleep patterns across different demographic and lifestyle factors using SQL.

Prepare data for visualization and further analysis in Power BI.
